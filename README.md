# Simple Inventory System (OOP in Python)

This is a mini project built using **Object-Oriented Programming (OOP)** concepts in Python.

## Features
1. Product management
2. Inventory handling
3. Store summary
4. Operator overloading (`__add__`)
5. Magic methods (`__str__`)

 ## OOP Concepts Used
1. Classes & Objects
2. Encapsulation
3. Composition (HAS-A relationship)
4. Magic Methods
5. Operator Overloading
